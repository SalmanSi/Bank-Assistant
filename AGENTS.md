# Bank Assistant Project

This is a Bank Assistant project that uses product knowledge data to help users with banking queries.

## Project Structure
- `NUST Bank-Product-Knowledge.xlsx` - Bank product knowledge base containing product information
- `opencode.json` - OpenCode configuration

## Guidelines
- When you need to search documentation for libraries or frameworks, use `context7` tools.
- Always follow best practices for the chosen language and framework.
- Write clean, well-documented code with proper error handling.
- When working with banking data, ensure data accuracy and handle sensitive information carefully.
